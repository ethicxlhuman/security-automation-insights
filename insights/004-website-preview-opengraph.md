# Security Firm Social Preview Generator

**Prototype link**: https://sentinel-mdr-cyberse-a6ux.bolt.host/

## The Problem
Mid-market security firms lose credibility when sharing content:
- Broken social previews on LinkedIn/Twitter kill click-through rates
- Generic preview images undermine professional positioning
- Manual HTML editing for Open Graph tags is time-consuming
- Security buyers judge competency by presentation quality

## The Solution
Automated Open Graph meta tag implementation for cybersecurity firms:
- Professional branded previews on all social platforms
- Custom preview images for service pages, case studies, blog posts
- Proper metadata for LinkedIn, Twitter, Telegram, WhatsApp
- Zero manual HTML editing per content piece

## Use Cases
1. **Service Marketing**: MDR/SOC firms sharing service pages with professional previews
2. **Thought Leadership**: Security executives sharing blog posts on LinkedIn
3. **Case Studies**: GRC consultants distributing compliance success stories
4. **Sales Enablement**: Analysts sharing threat reports with proper branding

## Tech Stack
- Open Graph Protocol meta tags
- Twitter Card optimization
- Figma/Canva for preview image design (1200x630px)
- Spartan security firm aesthetic (black, minimal, professional)

## Target Audience
Mid-market cybersecurity companies ($1M-50M revenue):
- MDR/SOC providers
- GRC consulting firms
- Cloud security companies
- Offensive security teams

## Impact
- Higher click-through rates on shared content
- Stronger trust signals with enterprise buyers
- Professional presentation in B2B sales cycles
- Competitive differentiation vs. firms with broken previews
