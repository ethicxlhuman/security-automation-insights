# Voiceflow x Convocore: Branded AI Agent for Security Firms

A Voiceflow agent integrated with Convocore and deployed as a fully branded chat widget on a cybersecurity company website.

## The Problem

Security founders lose qualified prospects who visit their site outside business hours. No one is available to answer "what do you offer?" or route them to the right service. Generic chatbots look unprofessional and erode trust with security buyers.

## The Solution

Connect a Voiceflow agent to Convocore, then customize the widget: dark background, branded avatar, company color scheme. The agent handles inbound inquiries, surfaces service cards (MDR triage, SOC automation, compliance), and routes prospects to a booking link.

## Use Cases

- Qualify inbound leads across MDR, GRC, and Cloud Security subsegments
- Surface relevant service details without human intervention
- Book discovery calls 24/7 without manual follow-up

## Impact

Converts cold website visitors into booked calls. Eliminates the gap between "visited site" and "scheduled demo" with zero manual effort.
