# Drag-and-Drop Service Section Builder

A configurable page layout tool that lets cybersecurity companies reorder their service sections in real time, persist the layout locally, and reset to defaults on demand.

## Problem

Security company websites are built once and rarely updated. But different buyer personas (CISO vs SOC Manager vs Compliance Lead) respond to different section orders. Restructuring a static page requires a developer, a deployment, and a delay. Most founders never do it. They lose the sale because the wrong section showed up first.

## Solution

A drag-and-drop builder with live preview that treats each service section as an ordered array item. Reorder on the left, see the result on the right. State persists across sessions via localStorage. Reset button restores original order when needed.

## Use Cases

- Security founders A/B testing MDR vs GRC section priority per campaign
- Proposal landing pages where service order matches prospect's stated pain point
- Internal demo environments where section emphasis shifts per prospect call

## Impact

Removes developer dependency from page structure decisions. Turns layout control into a founder-level action, not a sprint ticket.
